## 🔧 详细技术解析

### 1. Next.js 16 + App Router

**核心特性**：
- 服务端组件（Server Components）
- 流式渲染（Streaming）
- 混合渲染（SSR + SSG + ISR）
- 内置路由系统
- API Routes

**项目中的使用**：

```tsx
// src/app/(dashboard)/page.tsx
export default async function DashboardPage() {
  const { userId, orgId } = await auth();  // 服务端获取认证

  if (!userId) {
    redirect('/sign-in');  // 服务端重定向
  }

  prefetch(trpc.voices.getAll.queryOptions());  // 服务端预取数据

  return (
    <HydrateClient>  // 水合客户端
      <DashboardView />
    </HydrateClient>
  );
}
```

**关键概念**：
- **Server Components**: 默认组件类型，在服务端渲染，减少客户端 bundle
- **Client Components**: 使用 `'use client'` 标记，支持 hooks 和事件
- **Streaming**: 使用 Suspense 分块加载内容
- **Prefetching**: 服务端预取数据，避免客户端额外请求

### 5. Prisma ORM

**Schema 定义**：

```prisma
// prisma/schema.prisma
datasource db {
  provider = "postgresql"
}

generator client {
  provider = "prisma-client"
  output   = "../src/generated/prisma"
}

model Voice {
  id        String   @id @default(cuid())
  orgId     String?
  name      String
  variant   VoiceVariant
  category  VoiceCategory @default(GENERAL)
  language  String        @default("en-US")

  generations Generation[]

  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt

  @@index([orgId])
  @@index([variant])
}
```

**查询使用**：

```tsx
// 基本查询
const voices = await prisma.voice.findMany({
  where: { orgId: ctx.orgId },
  orderBy: { createdAt: 'desc' },
  select: { id: true, name: true, category: true },
});

// 关联查询
const voiceWithGenerations = await prisma.voice.findUnique({
  where: { id: voiceId },
  include: { generations: true },
});

// 事务
// $transaction 意思是将查询 tie 在一起，要么一起 succes,要么一起 失败
await prisma.$transaction([
  prisma.voice.create({ data: {...} }),
  prisma.generation.update({ where: {...}, data: {...} }),
]);

// 事务进阶
// 后异步操作可以用前异步操作的结果
await prisma.$transaction(async (tx) => {
  const voice = await tx.voice.create({ data: {...} });

  await tx.generation.update({
    where: {...},
    data: {
      voiceId: voice.id
    }
  });
});
```

### 6. tRPC + React Query

**tRPC 路由器**：

```tsx
// src/trpc/routers/voices.ts
export const voicesRouter = createTRPCRouter({
  getAll: orgProcedure
    .input(z.object({ query: z.string().optional() }).optional())
    .query(async ({ ctx, input }) => {
      const [custom, system] = await Promise.all([
        prisma.voice.findMany({ where: { variant: 'CUSTOM', orgId: ctx.orgId } }),
        prisma.voice.findMany({ where: { variant: 'SYSTEM' } }),
      ]);
      return { custom, system };
    }),

  delete: orgProcedure
    .input(z.object({ id: z.string() }))
    .mutation(async ({ ctx, input }) => {
      await prisma.voice.delete({ where: { id: input.id } });
      return { success: true };
    }),
});
```

**React Query 集成**：

```tsx
// 客户端使用
function VoicesView() {
  const trpc = useTRPC();

  // 查询
  const { data, isLoading } = useSuspenseQuery(
    trpc.voices.getAll.queryOptions({ query: 'test' })
  );

  // Mutation
  const deleteMutation = useMutation(
    trpc.voices.delete.mutationOptions({
      onSuccess: () => {
        queryClient.invalidateQueries({
          queryKey: trpc.voices.getAll.queryKey(),
        });
      },
    })
  );

  return (
    <Button onClick={() => deleteMutation.mutate({ id: '123' })}>
      Delete
    </Button>
  );
}
```

**queryOptions 优势**：
- 类型安全的查询键
- 自动推断输入和输出
- 支持 SSR 预取

**上传音频**：

```ts
export async function uploadAudio({
  buffer,
  key,
  contentType = 'audio/wav',
}: {
  buffer: Buffer;
  key: string;
  contentType?: string;
}): Promise<void> {
  await r2.send(
    new PutObjectCommand({
      Bucket: env.R2_BUCKET_NAME,
      Key: key,
      Body: buffer,
      ContentType: contentType,
    }),
  );
}
```

**获取签名 URL**：

```ts
import { GetObjectCommand } from '@aws-sdk/client-s3';
import { getSignedUrl } from '@aws-sdk/s3-request-presigner';

export async function getSignedAudioUrl(key: string): Promise<string> {
  const command = new GetObjectCommand({
    Bucket: env.R2_BUCKET_NAME,
    Key: key,
  });
  return getSignedUrl(r2, command, { expiresIn: 3600 }); // 1 小时
}
```

### 10. Wavesurfer.js 音频可视化

```tsx
const wavesurfer = WaveSurfer.create({
  container: '#waveform',

  barGap: 3,
  height: 120,

wavesurfer.play();
```

**React Hook 封装**：

```tsx
function useWaveSurfer(audioUrl: string) {
  const containerRef = useRef<HTMLDivElement>(null);
  const [isPlaying, setIsPlaying] = useState(false);

  useEffect(() => {
    const ws = WaveSurfer.create({
      container: containerRef.current,
      waveColor: 'rgb(200, 200, 200)',
      progressColor: 'rgb(100, 100, 200)',
      height: 120,
    });

    ws.load(audioUrl);
    ws.on('play', () => setIsPlaying(true));
    ws.on('pause', () => setIsPlaying(false));

    return () => ws.destroy();
  }, [audioUrl]);

  return { containerRef, isPlaying };
}
```

# Resonance - 需求 2：语音克隆 - 创建自定义语音

## 📝 需求描述

允许用户通过上传音频文件或录制音频，创建可复用的自定义语音。支持语音分类、多语言、描述信息等元数据。

---

## 🎯 核心功能

### 2.1 音频输入
- **文件上传**：支持所有音频格式，最大 20MB
- **麦克风录制**：实时录制，支持暂停/继续
- **音频验证**：最小时长 10 秒，验证音频格式

### 2.2 元数据填写
- **语音名称**：必填，用于标识语音
- **语音分类**：12 种预定义分类（General, Podcast, Voiceover 等）
- **语言选择**：支持所有语言和地区（en-US, zh-CN 等）
- **语音描述**：可选，描述语音特征和适用场景

### 2.3 数据持久化
- 音频文件存储到 Cloudflare R2
- 语音信息存入 PostgreSQL（Prisma）
- 记录创建者（orgId）用于多租户隔离

### 1. 数据模型

**`prisma/schema.prisma`**
```prisma
enum VoiceVariant {
  SYSTEM  # 系统预设语音
  CUSTOM  # 用户自定义语音
}

enum VoiceCategory {
  AUDIOBOOK
  CONVERSATIONAL
  CUSTOMER_SERVICE
  GENERAL
  NARRATIVE
  CHARACTERS
  MEDITATION
  MOTIVATIONAL
  PODCAST
  ADVERTISING
  VOICEOVER
  CORPORATE
}

model Voice {
  id String @id @default(cuid())

  orgId String?  # CUSTOM 语音必填，SYSTEM 为 null

  name        String
  description String?
  category    VoiceCategory @default(GENERAL)
  language    String        @default("en-US")
  variant     VoiceVariant
  r2ObjectKey String?  # 音频文件在 R2 的路径

  generations Generation[]  # 关联的生成记录

  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt

  @@index([variant])
  @@index([orgId])
}
```

```tsx
"use client";

import { useForm } from "@tanstack/react-form";
import { z } from "zod";
import { useDropzone } from "react-dropzone";

// 表单验证规则
const voiceCreateFormSchema = z.object({
  name: z.string().min(1, "Name is required"),
  file: z.instanceof(File).nullable().refine((f) => f !== null,
    "An audio file is required"),
  category: z.string().min(1, "A category is required"),
  language: z.string().min(1, "A language is required"),
  description: z.string(),
});

export function VoiceCreateForm() {
  const form = useForm({
    defaultValues: {
      name: "",
      file: null as File | null,
      category: "GENERAL",
      language: "en-US",
      description: "",
    },
    validators: {
      onSubmit: voiceCreateFormSchema,
    },
    onSubmit: async ({ value }) => {
      // 提交到后端
      const formData = new FormData();
      formData.append("file", value.file!);

      const params = new URLSearchParams({
        name: value.name,
        category: value.category,
        language: value.language,
        description: value.description,
      });

      const response = await fetch(`/api/voices/create?${params}`, {
        method: "POST",
        body: value.file,
      });

      if (response.ok) {
        toast.success("Voice created successfully!");
        form.reset();
      }
    },
  });

  return (
    <form onSubmit={(e) => { e.preventDefault(); form.handleSubmit(); }}>
      {/* 文件上传/录制 */}
      <form.Field name="file">
        {(field) => <FileDropzone {...field} />}
      </form.Field>

      {/* 语音名称 */}
      <form.Field name="name">
        {(field) => <Input {...field} placeholder="Voice Label" />}
      </form.Field>

      {/* 分类选择 */}
      <form.Field name="category">
        {(field) => (
          <Select value={field.state.value} onValueChange={field.handleChange}>
            {VOICE_CATEGORIES.map((cat) => (
              <SelectItem key={cat} value={cat}>
                {VOICE_CATEGORY_LABELS[cat]}
              </SelectItem>
            ))}
          </Select>
        )}
      </form.Field>

      {/* 语言选择 */}
      <form.Field name="language">
        {(field) => <LanguageCombobox {...field} />}
      </form.Field>

      {/* 描述 */}
      <form.Field name="description">
        {(field) => <Textarea {...field} placeholder="Describe this voice..." />}
      </form.Field>

      <Button type="submit">Create Voice</Button>
    </form>
  );
}
```

```tsx
function FileDropzone({ file, onFileChange, isInvalid }) {

        </div>

      </div>

  return (
    <div {...getRootProps()} className={cn(
      "border-2 border-dashed rounded-lg p-8",
      isDragActive && "border-primary",
      isDragReject && "border-destructive"
    )}>
      <input {...getInputProps()} />
      <p>Drag & drop audio file here, or click to select</p>
      <p className="text-sm">Max 20MB, min 10 seconds</p>
    </div>
  );
```

### 4. 音频录制组件

**`src/features/voices/components/voice-recorder.tsx`**

```tsx
"use client";

import RecordRTC from "recordrtc";
import { useState, useRef } from "react";

export function VoiceRecorder({ file, onFileChange }) {
  const [isRecording, setIsRecording] = useState(false);
  const [recorder, setRecorder] = useState<RecordRTC | null>(null);
  const mediaStreamRef = useRef<MediaStream | null>(null);

  const startRecording = async () => {
    const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
    mediaStreamRef.current = stream;

    const recorder = new RecordRTC(stream, {
      type: "audio",
      mimeType: "audio/wav",
      timeSlice: 5000,  // 每 5 秒保存一次
    });

    recorder.startRecording();
    setRecorder(recorder);
    setIsRecording(true);
  };

  const stopRecording = () => {
    if (recorder) {
      recorder.stopRecording(() => {
        const blob = recorder.getBlob();
        const file = new File([blob], `recording-${Date.now()}.wav`, {
          type: "audio/wav",
        });
        onFileChange(file);
      });

      if (mediaStreamRef.current) {
        mediaStreamRef.current.getTracks().forEach(track => track.stop());
      }

      setIsRecording(false);
    }
  };

  return (
    <div>
      {isRecording ? (
        <Button onClick={stopRecording}>
          <StopIcon /> Stop Recording
        </Button>
      ) : (
        <Button onClick={startRecording}>
          <MicIcon /> Start Recording
        </Button>
      )}
    </div>
  );
}
```

### 5. 后端 API

**`src/app/api/voices/create/route.ts`**

```ts
import { auth } from "@clerk/nextjs/server";
import { parseBuffer } from "music-metadata";
import { z } from "zod";
import { prisma } from "@/lib/db";
import { uploadAudio } from "@/lib/r2";
import { polar } from "@/lib/polar";

const createVoiceSchema = z.object({
  name: z.string().min(1),
  category: z.string().min(1),
  language: z.string().min(1),
  description: z.string().nullish(),
});

const MAX_UPLOAD_SIZE_BYTES = 20 * 1024 * 1024;  // 20MB
const MIN_AUDIO_DURATION_SECONDS = 10;

export async function POST(request: Request) {
  // 1. 验证认证
  const { userId, orgId } = await auth();
  if (!userId || !orgId) {
    return Response.json({ error: "Unauthorized" }, { status: 401 });
  }

  // 2. 验证订阅
  try {
    const customerState = await polar.customers.getStateExternal({
      externalId: orgId,
    });
    const hasActiveSubscription =
      (customerState.activeSubscriptions ?? []).length > 0;
    if (!hasActiveSubscription) {
      return Response.json({ error: "SUBSCRIPTION_REQUIRED" }, { status: 403 });
    }
  } catch {
    return Response.json({ error: "SUBSCRIPTION_REQUIRED" }, { status: 403 });
  }

  // 3. 验证查询参数
  const url = new URL(request.url);
  const validation = createVoiceSchema.safeParse({
    name: url.searchParams.get("name"),
    category: url.searchParams.get("category"),
    language: url.searchParams.get("language"),
    description: url.searchParams.get("description"),
  });

  if (!validation.success) {
    return Response.json({ error: "Invalid input" }, { status: 400 });
  }

  // 4. 读取音频文件
  const fileBuffer = await request.arrayBuffer();
  if (!fileBuffer.byteLength) {
    return Response.json({ error: "Please upload an audio file" }, { status: 400 });
  }

  if (fileBuffer.byteLength > MAX_UPLOAD_SIZE_BYTES) {
    return Response.json({ error: "Audio file exceeds the 20 MB size limit" },
      { status: 413 });
  }

  // 5. 验证音频格式和时长
  const contentType = request.headers.get("content-type") || "audio/wav";
  const normalizedContentType = contentType.split(";")[0]?.trim() || "audio/wav";

  let duration: number;
  try {
    const metadata = await parseBuffer(
      new Uint8Array(fileBuffer),
      { mimeType: normalizedContentType },
      { duration: true },
    );
    duration = metadata.format.duration ?? 0;
  } catch {
    return Response.json({ error: "File is not a valid audio file" }, { status: 422 });
  }

  if (duration < MIN_AUDIO_DURATION_SECONDS) {
    return Response.json({
      error: `Audio too short (${duration.toFixed(1)}s). Minimum duration is ${MIN_AUDIO_DURATION_SECONDS} seconds.`
    }, { status: 422 });
  }

  // 6. 创建语音记录
  let createdVoiceId: string | null = null;
  try {
    const voice = await prisma.voice.create({
      data: {
        name: validation.data.name,
        variant: "CUSTOM",
        orgId,
        description: validation.data.description,
        category: validation.data.category as any,
        language: validation.data.language,
      },
      select: { id: true },
    });

    createdVoiceId = voice.id;
    const r2ObjectKey = `voices/orgs/${orgId}/${voice.id}`;

    // 7. 上传音频到 R2
    await uploadAudio({
      buffer: Buffer.from(fileBuffer),
      key: r2ObjectKey,
      contentType: normalizedContentType,
    });

    // 8. 更新语音记录，保存 R2 路径
    await prisma.voice.update({
      where: { id: voice.id },
      data: { r2ObjectKey },
    });

    // 9. 上报用量事件到 Polar（异步，不阻塞响应）
    polar.events.ingest({
      events: [{
        name: "voice_creation",
        externalCustomerId: orgId,
        metadata: {},
        timestamp: new Date(),
      }],
    }).catch(() => {});

    return Response.json({ name: validation.data.name, message: "Voice created successfully" },
      { status: 201 });

  } catch (error) {
    // 失败时回滚：删除已创建的记录
    if (createdVoiceId) {
      await prisma.voice.delete({ where: { id: createdVoiceId } }).catch(() => {});
    }
    return Response.json({ error: "Failed to create voice" }, { status: 500 });
  }
}
```

### 6. R2 上传工具

**`src/lib/r2.ts`**

```ts
import { S3Client, PutObjectCommand } from "@aws-sdk/client-s3";
import { env } from "./env";

const r2 = new S3Client({
  region: "auto",
  endpoint: `https://${env.R2_ACCOUNT_ID}.r2.cloudflarestorage.com`,
  credentials: {
    accessKeyId: env.R2_ACCESS_KEY_ID,
    secretAccessKey: env.R2_SECRET_ACCESS_KEY,
  },
});

export async function uploadAudio({
  buffer,
  key,
  contentType = "audio/wav",
}: {
  buffer: Buffer;
  key: string;
  contentType?: string;
}): Promise<void> {
  await r2.send(
    new PutObjectCommand({
      Bucket: env.R2_BUCKET_NAME,
      Key: key,
      Body: buffer,
      ContentType: contentType,
    }),
  );
}
```

## 📊 数据流

```
1. 用户打开创建对话框
   ↓
2. 选择上传或录制音频
   - 上传：使用 react-dropzone
   - 录制：使用 RecordRTC
   ↓
3. 填写元数据（名称、分类、语言、描述）
   ↓
4. 提交表单
   ↓
5. 前端验证（Zod schema）
   ↓
6. 发送 POST /api/voices/create?name=...&category=...
   ↓
7. 后端验证：
   - 认证状态（Clerk）
   - 订阅状态（Polar）
   - 音频格式和时长（music-metadata）
   ↓
8. 创建 Voice 记录（Prisma）
   ↓
9. 上传音频到 R2
   ↓
10. 更新 Voice 记录（保存 r2ObjectKey）
   ↓
11. 上报用量事件到 Polar（异步）
   ↓
12. 返回成功，刷新语音列表
```

## ⚠️ 注意事项

1. **订阅检查**：创建语音前必须检查订阅状态
2. **错误处理**：失败时需要回滚（删除已创建的数据库记录）

4. **音频时长**：最小 10 秒，使用 music-metadata 验证

# Nuqs Tutorial 学习大纲

## 课程概述

nuqs 是一个类型安全的 URL 查询参数状态管理器，专门为 React 框架设计。它的核心思想是将 React 应用的状态存储在 URL 查询字符串中，让 URL 成为状态的单一来源。

## 核心概念回顾

- **Router**: API 路由容器，用于组织相关的端点
- **Procedure**: 具体的 API 端点，分为 query（查询）和 mutation（变更）
- **Context**: 请求上下文，用于传递认证信息等
- **Middleware**: 中间件，用于添加认证、日志等功能
- **Zod**: TypeScript 优先的输入验证库
- **Superjson**: 支持特殊数据类型的数据转换库
